# -Client-Developer-Website-Review-SM
Professional dual-perspective website auditing skill that analyzes websites as both a senior technical developer and a real customer. Detects technologies, evaluates UX/UI, SEO, performance, security, conversion paths, and generates client-ready reports, developer action plans, and presentation prompts.

Full Description

Client + Developer Website Review SM is a comprehensive website auditing skill designed for AI coding agents such as Claude Code, Codex, Cursor, Windsurf, and OpenCode.

The skill performs a complete website evaluation from two independent perspectives:

1. Senior Technical Auditor
    * Detects frameworks, CMS platforms, programming languages, hosting clues, analytics tools, tracking systems, APIs, SEO issues, performance bottlenecks, security concerns, accessibility problems, and mobile responsiveness.
2. Real Customer Auditor
    * Simulates a real visitor journey and evaluates usability, clarity, trust, navigation, content quality, conversion flow, visual appeal, and overall user experience.
